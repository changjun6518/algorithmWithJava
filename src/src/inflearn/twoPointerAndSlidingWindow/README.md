[공통원소구하기](공통원소구하기.java)
```
while (p1 < n && p2 < m) {
    if (a[p1] == b[p2]) {
        answer.add(a[p1]);
        p1++;
        p2++;
    } else if (a[p1] < b[p2]) {
        p1++;
    } else {
        p2++;
    }
}
```
